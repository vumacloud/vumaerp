# VumaERP Logo Assets

Replace these placeholder files with your actual brand assets:

## Required Files

| File | Size | Usage |
|------|------|-------|
| `logo.png` | 200x60px | Login page, light backgrounds |
| `logo.svg` | Vector | Scalable version |
| `logo_white.png` | 200x60px | Dark backgrounds |
| `logo_white.svg` | Vector | Navbar, dark backgrounds |
| `logo_small.png` | 40x40px | Footer, small spaces |
| `favicon.ico` | 32x32px | Browser tab icon |

## Color Palette

- Primary: `#1a5f7a` (Deep teal)
- Secondary: `#159895` (Lighter teal)
- Accent: `#57c5b6` (Light teal)

## Tips

1. Use transparent PNG backgrounds
2. Keep logos simple for small sizes
3. Test visibility on both light and dark backgrounds
4. Favicon should be recognizable at 16x16px
