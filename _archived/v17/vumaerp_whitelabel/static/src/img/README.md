# VumaERP Logo Assets

Replace these placeholder files with your actual branded assets:

## Required Files:

### logo.png
- **Dimensions:** 200x60 pixels (or similar aspect ratio)
- **Format:** PNG with transparent background
- **Usage:** Main logo on login page, navbar, emails

### logo_email.png
- **Dimensions:** 200x50 pixels
- **Format:** PNG (can have white background for email compatibility)
- **Usage:** Email headers and footers

### favicon.ico
- **Dimensions:** 32x32 pixels (multi-size ICO recommended: 16x16, 32x32, 48x48)
- **Format:** ICO
- **Usage:** Browser tab icon

### icon.png
- **Dimensions:** 180x180 pixels
- **Format:** PNG
- **Usage:** App icon, PWA icon

### banner.png (in /static/description/)
- **Dimensions:** 560x280 pixels
- **Format:** PNG
- **Usage:** Module description banner in Apps menu

## Color Palette:

- **Primary:** #714B67 (Purple/Mauve)
- **Secondary:** #017E84 (Teal)
- **Dark:** #21313C (Dark Blue-Gray)
- **Light:** #F8F9FA (Off-White)

## Logo Guidelines:

1. Keep text readable at small sizes
2. Ensure good contrast on both light and dark backgrounds
3. Test on login page, navbar, and email templates
4. Consider creating light and dark versions if needed
