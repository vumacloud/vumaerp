KRA e-TIMS Integration for Odoo v18

Complete Kenya Revenue Authority e-TIMS OSCU Integration

Overview

This module provides comprehensive integration with Kenya Revenue Authority's e-TIMS (Tax Invoice Management System) through the OSCU (Online Sales Control Unit) interface for Odoo v18 Community Edition.

Key Features

Device Management

Register and manage multiple OSCU devices with easy initialization and synchronization.

Invoice Submission

Submit customer invoices to e-TIMS automatically or manually with full compliance.

Product Registration

Register products with KRA e-TIMS system including item classifications and tax types.

QR Code Generation

Generate e-TIMS compliant receipts with QR codes for verification.

Additional Features

  • Item Classification Management
  • Tax Type Configuration
  • Credit Note Support
  • Multi-Company Support
  • Status Tracking
  • Comprehensive Error Handling

Easy Configuration

Simple setup process:

  1. Enable e-TIMS in Company Settings
  2. Initialize your OSCU device
  3. Sync item classifications from KRA
  4. Configure products and start invoicing

Requirements

  • Odoo v18 Community Edition
  • om_account_accountant module
  • Valid KRA e-TIMS registration
  • OSCU device serial number
