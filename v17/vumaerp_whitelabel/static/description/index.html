VumaERP White Label Module

Complete Rebranding Solution
Replace all Odoo references with your VumaERP branding throughout the entire application.

Features

  • Login page branding with custom logo
  • Web client navbar and interface branding
  • Email templates with VumaERP footer
  • Portal and website footer replacements
  • Database manager branding
  • Browser tab title and favicon
  • Dynamic JavaScript text replacements
  • Custom color scheme

What Gets Replaced

  • All "Odoo" text → "VumaERP"
  • Powered by footers → "Powered by VumaCloud"
  • Login page branding
  • Email signatures and footers
  • Portal welcome messages
  • Error page branding
  • About dialogs

Configuration

After installation:

  1. Replace logo files in /static/src/img/
  2. Customize colors in primary_variables.css
  3. Update URLs in ir_config_parameter_data.xml

Technical Details

  • Uses template inheritance for clean overrides
  • JavaScript MutationObserver for dynamic content
  • CSS-based text replacements where possible
  • Fully compatible with Odoo 17.0 CE
